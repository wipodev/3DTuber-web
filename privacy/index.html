3D Tuber

Privacy Policy

Last Updated: March 13, 2026

3D Tuber is a minimalist desktop application built with Electron, designed exclusively for content creator's production workflow. This application integrates the Snap Camera Kit SDK to provide augmented reality (AR) experiences.

1. Information Collection and Use

3D Tuber does not collect, store, or transmit any personal data to external servers.

  • Camera Access: The application requires access to the device's camera to render the AR Lens. The video feed is processed locally and in real-time.
  • No Personal Identifiable Information (PII): We do not collect names, email addresses, or any biographical data.
  • No Facial Recognition Storage: While the Snap Camera Kit technology processes facial features to apply AR effects, this data is processed locally on the device and is never stored or shared by this application.

2. Third-Party Services

This application uses the Snap Camera Kit SDK provided by Snap Inc. By using this application, the processing of camera data for AR effects is governed by Snap's Privacy Policy. 3D Tuber does not provide any user data to Snap Inc. beyond what is technically required for the SDK to function.

3. Data Security

Since no data is collected or stored, there is no risk of user data breach from our application.

4. Contact Us

If you have any questions about this Privacy Policy, you can contact the developer via email at ajwipo@gmail.com.
